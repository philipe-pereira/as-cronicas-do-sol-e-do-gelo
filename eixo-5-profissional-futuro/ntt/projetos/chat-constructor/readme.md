## **Visão Geral do Aplicativo**

Chat Constructor é uma plataforma que permite criar **ChatGPTs personalizados** utilizando **bases privadas de conhecimento**, organizadas em **blocos**.

Cada bloco contém arquivos (PDF, DOCX, TXT) que formam o contexto da IA. O administrador pode adicionar colaboradores, controlar permissões e publicar **URLs externas** para que **usuários finais** utilizem o chat sem acesso ao painel.

Quando um usuário faz uma pergunta através de uma URL externa, o sistema aciona a **API da OpenAI** adicionando apenas o conteúdo do **bloco correspondente** ao prompt, garantindo **segurança, personalização e isolamento de contexto**.
